# Cheesecake Front-end Challenge (Cupcaker)

This is a news site rendered using ReactJS with redux that loads the content from an API that uses express.js. The interface provides filtering capabilities to the user according to the selected category. 

It uses the languages HTML, CSS and JavaScript. It also uses Redux for state management, Axios to make the requests to the api, redux-thunk to async requests via Redux actions, React as front-end rendering engine and Express as backend API.

### Prerequisites

You will need nodeJS installed.

### Steps
Clone this repository
Run
```
  cd ckl-1_frontend_jessica_souza
  npm run init 
  npm run start
```
